Variable :

variable means the object values was changed . Eg:Flipcart product.

Rules for Variable name:
  • Var name are case sensitive: 'a'and'A' both are different.
  • Only letters,Digits,UnderScore(_),and $ allowed (not even space allowed).
  • Only letter Underscore(_) and $ allowed they should be a first character.
  • Reserved words can not be variable names.()
Keywords for declaring Variable:
  1. var

    Variable can re-declare and updated.It's Global scope variable,we can not use this keyword.

  2. let

    Variable can not be re-declare but can be updated.It's Block scope variable. Note:let can not be updated.

  3. const

    Variable can not be re-declared and updated.It's Block scope variable. Note:const object key values can be updated or changed .
